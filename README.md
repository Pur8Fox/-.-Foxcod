#Foxcod
Foxcod - преобразует русскую речь в python код. Пишите в блокнот программа.foxcod и запускаете через cmd используя ключ "fox программа.foxcod".
  Foxcod - личный язык програмиста созданный специально для облегчения написания кодинга на питоне и некоторых задач. этим языком могут пользоваться все, но ещё требуется доработка кода для более качественного создание ui. как он работает: есть мозг "engine.py" он парсит команды, принимает решения, вызывает действия (UI, файлы, система), есть мысль "программа.foxcod" описание задачи, сценарий, инструкция, есть мотивация "cmd" cmd запускает fox, fox запускает engine.py, engine.py читает test.foxcode, выполняет команды, cmd просто показывает лог выполнения.
  Фокскод преобразует естественный ввод в системные операции. это должно быть приложением как питон чтобы ты мог удобно кодить.
Правила синтаксиса:

    Объекты: Текст в кавычках "" — это названия файлов или папок.
    Команды: Текст без кавычек — это действия (операторы).
    Маркеры статуса (ПИШУТСЯ ВСЕГДА В САМОМ НАЧАЛЕ СТРОКИ):
        ! — Ошибка, опечатка, отсутствие файла или запрет доступа.
        ? — Вопрос к пользователю или выбор из нескольких вариантов.
        √ — Всё успешно.

Примеры работы (Dataset):

    Ввод: открыть "папка 1" — Вывод: "папка 1" открыто
    Ввод: отлыть "папка 1" — Вывод: !опечатка в местах "отлыть"
    Ввод: открыть "папка 2" — Вывод: !"папка 2" не найдено
    Ввод: открыть "папка 1" (если их три) — Вывод: ?найдено 3 папки "папка 1", какую открыть?
    Ввод: открыть "Фото", найти "отпуск.jpg" и скопировать в "Архив" — Вывод: задачи успешно выполнены
    Ввод: [цепочка с ошибкой в конце] — Вывод: !задачи выполнены, но "архив" не перенесён т.к. "папка 1" не существует
    Ввод: открыть "папка 1" -> удали её — Вывод: "папка 1" открыто -> "папка 1" удалено (использование контекста)
    Ввод: удалить всё в "System32" — Вывод: !не удалось т.к. это системная папка, удаление заблокированно
    Ввод: сколько весит "Видео"? — Вывод: 15Гб
    Ввод: удали все ".txt" в "папка 1" — Вывод: удалены все ".txt" в "папка 1"
    Ввод: перенеси "фото" на рабочий стол — Вывод: папка "фото" перенесена на рабочий стол (понимание сокращенных путей)
    Ввод: зайди в "папка 1" -> выйди назад — Вывод: сделано -> вышел назад (навигация по истории)
    Ввод: путь к "диплом" — Вывод: C:/Документы/Университет/Финал/диплом (глобальный поиск)
    Ввод: добавить "D:/Музыка" в группу "Аудио" -> очистить "аудио" — Вывод: группа создана -> группа "аудио" очищено
