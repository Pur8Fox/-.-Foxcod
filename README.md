 Foxcod — Твой личный интерпретатор автоматизации
Фундамент: Python 3.14+
Забудь про сложный синтаксис, скобки и точки с запятой. Пиши команды так, как ты думаешь, и Foxcod исполнит их в один клик.
🚀 Быстрый старт

    Создай файл: script.foxcod
    Напиши в нём команду: открой файл "Документы"
    Запусти в консоли:
    bash

    fox script.foxcod

    Используйте код с осторожностью.

🛠 Основы языка
В Foxcod всё просто:

    "Текст в кавычках" — это объекты (файлы, папки, названия).
    Текст без кавычек — это команды (действия).
    Контекст (->) — Foxcod помнит, с чем ты работал последним.

🚦 Маркеры статуса (всегда в начале строки):

    √ — Всё прошло успешно.
    ? — Нужно уточнение (например, если найдено несколько файлов).
    ! — Ошибка (опечатка, файл не найден или доступ запрещён).

💡 Примеры команд
Команда	Результат
открыть "папка 1"	√ "папка 1" открыто
открыть "Фото" -> удали её	Откроет и сразу удалит (работает контекст)
сколько весит "Видео"?	15 Гб
путь к "диплом"	Глобальный поиск файла по всему ПК
удали все ".txt" в "папка 1"	Массовая очистка по расширению
удалить всё в "System32"	! Ошибка: удаление системных папок заблокировано
🧠 Интеллектуальные возможности

    Понимание сокращений: Просто напиши "на рабочий стол", и Foxcod сам найдет путь.
    Навигация: Команда выйди назад вернет тебя в предыдущую папку.
    Группировка: добавить "D:/Музыка" в группу "Аудио" -> очистить "Аудио".
    Исправление опечаток: Если напишешь отлыть, Foxcod подсветит: !опечатка в слове "отлыть".
